
# Brain Tumor Segmentation using U-Net and PyTorch

## Overview

This project implements a deep learning pipeline for brain tumor segmentation from MRI images using U-Net architecture in PyTorch.

The project includes:
- Data preprocessing
- U-Net implementation
- Training pipeline
- Evaluation metrics
- Visualization tools
- Streamlit deployment
- ONNX export support

## Dataset

LGG MRI Segmentation Dataset:
https://www.kaggle.com/datasets/mateuszbuda/lgg-mri-segmentation

## Tech Stack

- Python
- PyTorch
- MONAI
- OpenCV
- Albumentations
- Streamlit
- TensorBoard
